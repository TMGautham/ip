9. VirtualDOM, Reconciliation, Diffing

VirtualDOM
The Virtual DOM (VDOM) is an in-memory representation of Real DOM. The representation of a UI is kept in memory and synced with the "real" DOM. 

Reconciliation
Reconciliation is the process of comparing the previous state of the DOM with the new state, identifying the differences, and updating only the necessary parts to reflect the changes. 

Diffing
React uses a diffing algorithm as part of its reconciliation process to efficiently update the Document Object Model (DOM). When the state of a component changes, React creates a new virtual DOM and compares it with the current DOM. This comparison process, known as â€œdiffing,â€ allows React to identify the minimum number of operations needed to update the DOM.
